# 🎴 漢字よみかた神経衰弱

小学生が漢字の読み方を楽しく学べる、ブラウザで遊べる神経衰弱ゲームです。

![スクリーンショット](screenshot.png)

## 🎮 遊び方

1. 学年（1〜6年生）とペア数（6〜12ペア）を選ぶ
1. **ゲームスタート！** を押す
1. カードをめくって **漢字** と **よみかな** のペアを見つけよう
1. 全部マッチさせたらクリア！✨

## ✨ 特徴

- **1〜6年生**の学習漢字に対応（各学年50語以上）
- **ペア数を選択**できるので難易度を調整できる（6・8・10・12ペア）
- めくった回数・かかった時間を記録してスコア評価
- スマートフォン・タブレットでも快適に遊べるレスポンシブデザイン
- サーバー不要・インターネット接続なしでも遊べる（Webフォント除く）

## 🚀 使い方

### オンラインで遊ぶ

> GitHub Pages でホストしている場合はここにURLを記載

### ローカルで動かす

```bash
git clone https://github.com/あなたのユーザー名/kanji-memory-game.git
cd kanji-memory-game
# index.html をブラウザで開くだけでOK！
open index.html   # macOS
start index.html  # Windows
```

> ⚠️ ローカルでCSSフォントが読み込まれない場合はブラウザのCORSポリシーが原因です。`npx serve .` などで簡易サーバーを起動してください。

## 📖 漢字データについて

`kanji-data.js` に小学校学習漢字（文部科学省 学習指導要領準拠）の読みデータが収録されています。  
データの追加・修正はこのファイルを編集するだけでOKです。

```js
// フォーマット例
{ kanji: "山", reading: "やま" },
```

## 🛠 カスタマイズ

### 漢字を追加する

`kanji-data.js` の該当する学年の配列に追記してください。

```js
const KANJI_DATA = {
  1: [
    { kanji: "一", reading: "いち" },
    { kanji: "新しい漢字", reading: "よみかな" }, // ← 追記
    ...
  ],
};
```

### 外見をカスタマイズする

`style.css` の `:root` セクションで色を変更できます。

```css
:root {
  --navy:   #1a1a5e;  /* ヘッダーの色 */
  --yellow: #ffd93d;  /* アクセントカラー */
  ...
}
```
